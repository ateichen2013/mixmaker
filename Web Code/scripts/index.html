Using Google forms for a survey on favorite baby names

Homework 2

Homework 2 was the first google survey I have ever created. The purpose of this assignment was to use Google forms to create a survey on favorite baby names. This form I created has a total of four questions, however, the user will only have to answer two. The first question determines what the second question will be. The first question asks for the gender the baby name is for and is a multiple-choice question. The options for the genders are male, female, and the gender-neutral. The second question then ask what the name is that you want to input and is a short answer response. Depending on what you choose for the first question your second question will be based off that gender. For instance, if someone choose male as the gender the second question will ask “what is your favorite male name for a baby?”. Once the user finishes the survey they can see the results for that survey. All survey entries are anonyms and does not ask for any personal information The audience of this survey is students from this class, given from posting the URL on a discussion page along with my Facebook friends also using a link I posted.
